#  Análisis de Factores del Desempeño Académico - Saber 11 (ICFES)

---

##  Descripción del proyecto
Este proyecto analiza los factores socioeconómicos, demográficos e institucionales que influyen en el desempeño académico de los estudiantes colombianos en las pruebas Saber 11.

Se utilizan datos abiertos del ICFES con el fin de identificar patrones y relaciones que permitan entender las brechas educativas en Colombia.

---

##  Objetivo
Identificar cómo variables como el estrato socioeconómico, la educación de los padres, hábitos de estudio y características de las instituciones afectan los resultados académicos.

---

##  Tecnologías utilizadas
- Python  
- requests  
- pandas  
- API de Datos Abiertos de Colombia  
- Google BigQuery (Data Warehouse)  
- SQL (consultas analíticas)  
- Power BI (visualización de datos)  

---

##  Pipeline de datos

###  Extracción
- Consumo de datos desde API pública  
- Integración de datos abiertos del ICFES  

###  Transformación
- Limpieza de datos  
- Manejo de valores nulos  
- Estandarización de variables  

###  Carga
- Almacenamiento en Google BigQuery  
- Modelado de datos para análisis  

###  Visualización
- Creación de dashboards interactivos en Power BI  
- Análisis de indicadores clave  

---

##  Análisis realizado
- Relación entre nivel socioeconómico y desempeño  
- Impacto de la educación de los padres  
- Influencia de hábitos de estudio  
- Comparación entre instituciones educativas  

---

## 📈 Resultados
- Identificación de factores clave en el rendimiento académico  
- Detección de brechas educativas  
- Generación de insights para toma de decisiones
